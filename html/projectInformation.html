Keylogger

Tipo de projecto: Personal.

Fecha: 5 Jul. 2017.

Lenguaje: C++.

Tecnologías: API de Windows.

Paradigma: Imperativo.

Descripción: Este es un programa que funciona de manera local el cual tiene como proposito capturar todas la teclas pulsadas y guadarlas en un archivo para su futura revisón. Este programa una vez que se ejecuta la primera vez se inicia automáticamente al iniciar Windows, función que se puede desactivar desde el administrador de tareas. Finalmente, se ejecuta en segundo plano para evitar molestias.
